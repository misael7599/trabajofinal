# Recetas de Comida

Pagina web sobre recetas caseras faciles y rapidas.

## Secciones

- Inicio: bienvenida y enlace externo
- Recetas: fideos con salsa y ensalada fresca
- Ingredientes: tabla con los ingredientes de cada receta
- Contacto: formulario para mandar tu receta

## Tecnologias usadas
